## CLR via C#

> 大致分三种读法：
>
> 细读：都要读懂，要都理解了，读不懂反复读，找额外资料读。
> 通读：大致都了解可以干嘛，尽量看懂。
> 粗读：随手翻下，读不懂可以跳过，时不时回头看看。
>
> 以第4版为例：
>
> Ch1通读。
> Ch2和3粗读。
> Ch4到19：细读，全是基础内容。
> Ch20细读，最后两节（CER和Code Contract）可以粗读。
> Ch21细读，讲GC的，比较重要。
> Ch22粗读。
> Ch23到25通读。
> Ch26细读。
> Ch27到30通读。
>
